# Get-OktaConnection

## SYNOPSIS
Gets details of the current Okta connection.

## SYNTAX


## DESCRIPTION

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-OktaConnection
```

Get the Okta connection details.
